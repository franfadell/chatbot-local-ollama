# Chatbot en Python

Chatbot desarrollado en Python como práctica de programación modular y manejo de datos.  
Permite responder consultas básicas utilizando información almacenada en un archivo JSON.  
El código es simple, fácil de entender y puede ampliarse sin dificultad.

## Características principales
- Lectura de información desde un archivo JSON  
- Código organizado en módulos  
- Interacción por consola  
- Proyecto ligero y educativo

## Tecnologías utilizadas
- Python 3.x  
- JSON  
- Programación modular

## Cómo ejecutar el proyecto
```
python main.py
```

O en PowerShell:
```
py main.py
```
